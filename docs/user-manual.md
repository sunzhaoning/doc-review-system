# 用户手册

## 概述

文档评审系统是一个专业的文档评审协作平台，支持文档上传、多人评审、意见收集和归档管理的完整流程。本手册将帮助您快速掌握系统的使用方法。

---

## 快速开始

### 登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击「登录」按钮

**首次登录**：请使用管理员分配的账号密码登录，登录后建议立即修改密码。

### 系统首页

登录成功后进入首页仪表盘，展示：

- **待办事项**：待评审文档数量、待处理任务
- **评审统计**：本周评审数、通过率统计
- **最近文档**：最近提交和评审的文档
- **系统公告**：管理员发布的公告信息

## 功能模块

### 1. 文档管理

#### 1.1 上传文档

1. 点击左侧菜单「文档管理」→「文档列表」
2. 点击右上角「上传文档」按钮
3. 填写文档信息：
   - **文档标题**：必填，简洁明了
   - **文档描述**：选填，详细说明文档内容
   - **评审类型**：选择评审类型
     - 技术评审 (TECHNICAL)
     - 设计评审 (DESIGN)
     - 代码评审 (CODE)
   - **截止日期**：选填，设置评审截止时间
   - **文档文件**：支持 PDF、Word、Excel、PPT 等格式，最大 50MB
4. 点击「确认上传」

**注意事项**：
- 上传后文档状态为「草稿」
- 草稿状态可继续编辑或删除
- 文件命名建议使用有意义的名称

#### 1.2 查看文档列表

文档列表展示所有有权限查看的文档：

- **筛选条件**：标题、状态、评审类型、提交者
- **状态说明**：
  - 📝 草稿：未提交评审
  - ⏳ 待评审：已提交，等待评审者处理
  - 🔍 评审中：评审者正在评审
  - 📝 修订中：需要修改后重新提交
  - ✅ 已通过：评审通过
  - ❌ 已拒绝：评审未通过
  - 📦 已归档：已归档保存

#### 1.3 查看文档详情

点击文档标题进入详情页，可查看：

- 文档基本信息
- 文档文件预览/下载
- 评审进度
- 评审意见列表
- 评论记录

#### 1.4 提交评审

1. 在文档详情页，点击「提交评审」按钮
2. 选择评审者（支持多选）
3. 设置评审截止日期（可选）
4. 确认提交

**注意**：提交后文档状态变为「待评审」，无法再编辑文档内容。

#### 1.5 撤回评审

在「待评审」状态下，提交者可以撤回评审：

1. 进入文档详情页
2. 点击「撤回评审」按钮
3. 确认撤回

撤回后文档恢复为「草稿」状态，可重新编辑。

### 2. 评审管理

#### 2.1 查看待评审文档

1. 点击左侧菜单「评审管理」→「待评审」
2. 查看分配给您的待评审文档列表
3. 列表显示：文档标题、评审类型、截止日期、提交者

#### 2.2 开始评审

1. 在待评审列表中，点击文档标题
2. 进入评审页面，可查看：
   - 文档基本信息
   - 文档文件（支持预览和下载）
   - 其他评审者的意见（如有）
3. 点击「开始评审」按钮，状态变为「评审中」

#### 2.3 填写评审意见

评审表单包含以下内容：

| 字段 | 说明 | 必填 |
|------|------|------|
| 评审决定 | 通过/拒绝/需要修订 | 是 |
| 总体评价 | 对文档的整体评价 | 是 |
| 优点 | 文档的优点和亮点 | 否 |
| 问题 | 发现的问题和不足 | 否 |
| 建议 | 改进建议 | 否 |

**评审决定说明**：
- ✅ **通过**：文档质量符合要求，可以继续下一步
- ❌ **拒绝**：文档存在严重问题，需要重新编写
- 📝 **需要修订**：文档基本可行，但需要修改部分内容

#### 2.4 提交评审结果

1. 填写完评审意见后，点击「提交评审」
2. 确认提交内容
3. 提交后不可修改

#### 2.5 查看评审历史

1. 点击左侧菜单「评审管理」→「评审历史」
2. 查看您已完成的所有评审记录
3. 支持按决定类型筛选

---

### 3. 评论功能

#### 3.1 添加评论

在文档详情页，可以对文档进行评论：

1. 点击「添加评论」按钮
2. 选择评论类型：
   - ❗ 问题：发现的问题
   - 💡 建议：改进建议
   - ❓ 疑问：需要澄清的问题
3. 设置优先级：高/中/低
4. 输入评论内容
5. （可选）填写位置信息，用于行内批注
6. 提交评论

#### 3.2 回复评论

1. 点击评论下的「回复」按钮
2. 输入回复内容
3. 提交回复

#### 3.3 编辑/删除评论

- 只能编辑/删除自己发表的评论
- 点击评论右上角的「编辑」或「删除」按钮

---

### 4. 归档管理

#### 4.1 查看归档文档

1. 点击左侧菜单「归档管理」
2. 搜索已归档的文档
3. 支持按关键词、评审类型、日期范围筛选

#### 4.2 归档文档

1. 在文档详情页，评审完成后点击「归档」
2. 确认归档操作
3. 归档后文档不可编辑

#### 4.3 批量归档

1. 在文档列表中，勾选多个文档
2. 点击「批量归档」按钮
3. 确认操作

#### 4.4 导出评审报告

1. 进入已归档文档详情页
2. 点击「导出报告」按钮
3. 下载 Markdown 格式的评审报告

报告包含：
- 文档基本信息
- 评审统计
- 所有评审意见汇总
- 评论记录

### 5. 系统管理

> 以下功能需要管理员权限

#### 5.1 用户管理

**用户列表**

1. 点击「系统管理」→「用户管理」
2. 查看所有用户，支持按用户名、姓名、状态筛选

**创建用户**

1. 点击「新增用户」按钮
2. 填写用户信息：
   - 用户名（唯一）
   - 密码
   - 真实姓名
   - 邮箱
   - 手机号
   - 所属部门
   - 分配角色
3. 保存

**编辑用户**

- 修改用户基本信息
- 分配/调整角色
- 启用/禁用账号
- 重置密码

**用户状态**：
- 🟢 启用：正常使用
- 🔴 禁用：无法登录

#### 5.2 角色管理

**角色概念**：角色是权限的集合，用户通过角色获得相应权限。

**预设角色**：

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 超级管理员 | 系统最高权限 | 所有权限 |
| 管理员 | 系统管理权限 | 用户管理、文档管理 |
| 评审员 | 文档评审权限 | 评审、评论 |
| 普通用户 | 基础权限 | 提交文档、查看 |

**创建角色**

1. 点击「新增角色」按钮
2. 填写角色信息：
   - 角色名称
   - 角色编码（唯一标识）
   - 角色描述
3. 分配菜单权限
4. 分配操作权限
5. 保存

**分配菜单权限**

菜单权限控制用户可以看到哪些菜单：

1. 在角色编辑页，切换到「菜单权限」标签
2. 勾选可访问的菜单
3. 保存

**分配操作权限**

操作权限控制用户可以执行哪些操作：

1. 在角色编辑页，切换到「操作权限」标签
2. 勾选允许的操作权限
3. 保存

#### 5.3 菜单管理

管理系统的菜单结构：

**菜单类型**：
- 📁 目录：菜单分组，不对应具体页面
- 📄 菜单：对应具体页面
- 🔘 按钮：页面内的操作按钮

**菜单属性**：
| 属性 | 说明 |
|------|------|
| 菜单名称 | 显示名称 |
| 菜单编码 | 唯一标识 |
| 上级菜单 | 父级菜单 |
| 路由地址 | 前端路由路径 |
| 组件路径 | Vue 组件路径 |
| 权限标识 | 权限编码 |
| 图标 | 菜单图标 |
| 排序 | 显示顺序 |
| 是否可见 | 是否在菜单中显示 |

**操作**：
- 新增：添加新菜单
- 编辑：修改菜单信息
- 删除：删除菜单（需先删除子菜单）

#### 5.4 权限管理

管理系统所有操作权限：

**权限类型**：
- 菜单权限：控制菜单访问
- 按钮权限：控制按钮操作
- API 权限：控制接口访问

**权限编码规则**：
```
模块:资源:操作

示例：
sys:user:list    - 用户列表
sys:user:create  - 创建用户
sys:user:edit    - 编辑用户
sys:user:delete  - 删除用户
doc:list         - 文档列表
doc:create       - 创建文档
review:submit    - 提交评审
```

#### 5.5 系统配置

**基础配置**

管理系统运行参数：

| 配置项 | 说明 |
|--------|------|
| 系统名称 | 系统显示名称 |
| 文件大小限制 | 上传文件最大限制 |
| 允许文件类型 | 允许上传的文件类型 |

**LDAP/AD 域配置**

配置企业 AD 域认证：

1. 进入「系统配置」→「LDAP 配置」
2. 填写配置信息：
   - LDAP 服务器地址
   - 基础 DN
   - 绑定账号
   - 绑定密码
   - 用户属性映射
3. 点击「测试连接」验证配置
4. 启用 LDAP 认证

**AD 域用户登录**：
- 启用后，用户使用域账号登录
- 首次登录自动创建本地用户记录

## 常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码。

### Q2: 文档上传失败？

可能原因：
1. 文件大小超过限制（50MB）
2. 文件格式不支持
3. 网络连接问题
4. 存储空间不足

解决方法：
- 检查文件大小和格式
- 检查网络连接
- 联系管理员检查存储配置

### Q3: 无法提交评审？

可能原因：
1. 没有评审权限
2. 文档状态不是「待评审」或「评审中」
3. 评审表单未完整填写

解决方法：
- 联系管理员确认权限
- 确认文档状态
- 检查必填字段是否已填写

### Q4: 看不到某些菜单？

可能原因：
- 当前角色没有对应菜单权限

解决方法：
- 联系管理员分配相应权限

### Q5: 如何修改个人信息？

1. 点击右上角用户头像
2. 选择「个人设置」
3. 修改个人信息
4. 保存

### Q6: 如何修改密码？

1. 点击右上角用户头像
2. 选择「修改密码」
3. 输入旧密码和新密码
4. 确认修改

### Q7: 文档预览显示异常？

可能原因：
- 浏览器不支持该文件类型预览
- 文件损坏

解决方法：
- 下载文件后使用本地软件打开
- 重新上传正确文件

### Q8: 如何批量操作？

在列表页面：
1. 勾选需要操作的项目
2. 点击批量操作按钮
3. 确认操作

支持批量操作：归档、删除等

### 文档评审流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  草稿   │───>│ 待评审  │───>│ 评审中  │───>│ 已通过  │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     │              │              │              │
     │              │              │              │
     v              v              v              v
   删除           撤回         需要修订         已拒绝
                               │                │
                               v                v
                            ┌─────────┐    ┌─────────┐
                            │ 修订中  │    │  已拒绝 │
                            └─────────┘    └─────────┘
                                 │
                                 v
                            重新提交
```
