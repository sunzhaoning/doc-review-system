## P2级别 - 建议优化

### BUG-001: 前端缺少路由守卫

| 字段 | 内容 |
|------|------|
| **ID** | BUG-001 |
| **级别** | P2-Minor |
| **模块** | 前端 |
| **发现时间** | 2026-03-06 |
| **状态** | 待修复 |
| **描述** | 前端路由配置缺少全局路由守卫，未登录用户可能直接访问需要认证的页面 |
| **复现步骤** | 1. 未登录状态<br>2. 直接访问 /dashboard 等页面 |
| **预期结果** | 自动跳转到登录页 |
| **实际结果** | 页面可能正常显示（取决于API返回） |
| **建议修复** | 在 router/index.ts 添加 beforeEach 导航守卫 |
| **影响范围** | 前端路由 |
| **修复优先级** | 低（API已有权限校验） |

### BUG-002: 缺少接口限流配置

| 字段 | 内容 |
|------|------|
| **ID** | BUG-002 |
| **级别** | P2-Minor |
| **模块** | 后端 |
| **发现时间** | 2026-03-06 |
| **状态** | 待修复 |
| **描述** | 系统缺少接口限流配置，可能被恶意请求攻击 |
| **复现步骤** | 1. 短时间大量请求同一接口 |
| **预期结果** | 应限制请求频率 |
| **实际结果** | 无限制 |
| **建议修复** | 添加 @RateLimiter 注解或 Redis 限流 |
| **影响范围** | 全部API |
| **修复优先级** | 低（生产环境建议配置） |
