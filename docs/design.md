# 文档评审系统 - 交互设计文档

**版本**: 1.0
**日期**: 2026-03-05
**作者**: ux-designer
**状态**: 待审核

---

## 1. 设计概述

### 1.1 设计原则

- **简洁高效**：面向研发人员，界面简洁，操作直接
- **角色分明**：不同角色看到不同功能入口
- **状态可见**：评审进度、文档状态一目了然
- **流程清晰**：上传→评审→归档，每步都有明确指引

### 1.2 设计风格

- **配色**：专业蓝灰色系，突出内容
- **字体**：系统默认字体，保证可读性
- **图标**：线性图标，简洁现代
- **布局**：左侧导航 + 右侧内容区

## 2. 用户角色与界面

### 2.1 角色视图矩阵

| 角色 | 默认首页 | 导航菜单 | 快捷操作 |
|------|----------|----------|----------|
| 提交者 | 我的文档 | 我的文档、待处理、归档检索 | 上传文档 |
| 评审者 | 待评审 | 待评审、我的评审、归档检索 | - |
| 管理员 | 控制台 | 控制台、文档管理、用户管理、系统设置 | 上传文档、归档管理 |
| 观察者 | 文档列表 | 文档列表、归档检索 | - |

## 5. 交互细节

### 5.1 状态流转

```
┌─────────────────────────────────────────────────────────────┐
│                        文档状态流转                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌───────┐      提交评审      ┌──────────┐                │
│   │ 草稿  │ ────────────────→ │ 待评审   │                │
│   └───────┘                    └──────────┘                │
│       ↑                              │                      │
│       │ 重新编辑                     │ 首位评审者开始       │
│       │                              ↓                      │
│   ┌──────────┐    请求修改    ┌──────────┐                │
│   │ 待修改   │ ←──────────── │ 评审中   │                │
│   └──────────┘               └──────────┘                │
│       │                           │    │                  │
│       │ 重新提交                   │    │ 满足通过条件     │
│       ↓                           │    │                  │
│   ┌──────────┐    不满足条件  ┌──────────┐                │
│   │ 待评审   │ ←──────────── │ 已拒绝   │                │
│   └──────────┘               └──────────┘                │
│                                    │                       │
│                                    │ 管理员归档           │
│                                    ↓                      │
│                              ┌──────────┐                │
│                              │ 已通过   │ ──→ 已归档     │
│                              └──────────┘                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 待修改 | 黄色 | ✏️ | 需要提交者修改 |
| 已通过 | 绿色 | ✓ | 评审通过 |
| 已拒绝 | 红色 | ✗ | 评审拒绝 |
